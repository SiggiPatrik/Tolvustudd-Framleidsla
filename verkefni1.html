Undirbúningur

Ég byrjaði á því að horfa kennslumyndböndin, þau voru hjálpleg með uppsetninguna. Þar að auki hef ég smá reynslu af HTML frá CS50 netnámskeiðinu sem Harvard býður upp á. Ég valdi að nota Brackets frekar en VS Code út af 'Live' eiginleikanum sem það býður upp á. Síðan náði ég mér einnig í Git Bash.

Loks hlóð ég niður Spectral vefsíðusniðmátinu frá Html5Up og byrjaði að slá inn þennan texta. Ég valdi Spectral því ég vildi ágætlega 'minimalistic look' og vildi frekar hafa dropdown sidebar frekar en header eða permanent sidebar. Því varð Spectral fyrir valinu. Öll skjöl sem tilheyra síðunni fóru í einn folder sem er vistaður locally. Síðan þarf að koma þessu öllu í GitHub skýið.

Uppsetning á GitHub

Ég bjó til nýtt repository á GitHubinu mínu sem heitir Tolvustudd-Framleidsla. Síðan opnaði ég git bash í local möppunni sem innihélt skjöl síðunnar. Svo keyri ég þessa skipun til að

Félagi minn og samnemandinn hann Sindri Dagur er flinkur í GitHub svo hann ráðlagði mér að nota eftirfarandi skipanir til að koma local foldernum á GitHub:

                                    git clone https://github.com/SiggiPatrik/Tolvustudd-Framleidsla
git add .
git commit -m "Setja template files inn"
git push

Þetta bara svínvirkaði í fyrstu tilraun svo ekki þurfti að hafa meiri áhyggjur af því. Þetta er svo gert í hvert sinn til að vista breytingar úr local folder yfir í GitHub repositoryið. Að sjálfsögðu er commentið ekki það sama að hverju sinni samt, heldur það sem er viðeigandi í hvert skipti.

Breyting á síðunni

Ég eyddi miklum tíma í að dunda mér við að snyrta sniðmátið og setja inn suma hluti og fjarlægja aðra. Ég mun líklegast aldrei klára það alveg en vonandi get ég talið útlit vefsíðunnar ásættanlegt að lokum.

Myndir og skjáskot

Það er nú enn ekki mikið um myndir á síðunni (allavega ekki enn). En þær myndir sem eru á síðunni voru ekki með neitt vesen og óþarfi var að smækka þær fyrir síðuna. Hins vegar er myndin hér ofan á síðunni með leiðindi því ég fæ hana ekki til að vera á miðri síðunni, sama hvað ég geri við css kóðann.

Væntingar mínar til áfangans

Ég vil að þessi áfangi hjálpi mér að skipuleggja og framkvæma töff verkefni sem ég get svo sett fram skipulega á þessa síða og notað hana sem part af umsóknum. Ég vil leggja mig fram í að gera þessi verkefni vel og sérstaklega að greina vel frá þeim svo þessi síða nýtist sem fjöður í hattinum mínum í framtíðinni.
