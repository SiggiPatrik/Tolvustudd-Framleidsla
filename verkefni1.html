Verkefni 1 í Tölvustuddri framleiðslu - Vefsíðugerð

Undirbúningur

Ég byrjaði á því að horfa kennslumyndböndin, þau voru hjálpleg með uppsetninguna. Þar að auki hef ég smá reynslu af HTML frá CS50 netnámskeiðinu sem Harvard býður upp á. Ég valdi að nota Brackets frekar en VS Code út af 'Live' eiginleikanum sem það býður upp á. Síðan náði ég mér einnig í Git Bash.

Loks hlóð ég niður Spectral vefsíðusniðmátinu frá Html5Up og byrjaði að slá inn þennan texta. Ég valdi Spectral því ég vildi ágætlega 'minimalistic look' og vildi frekar hafa dropdown sidebar frekar en header eða permanent sidebar. Því varð Spectral fyrir valinu. Öll skjöl sem tilheyra síðunni fóru í einn folder sem er vistaður locally. Síðan þarf að koma þessu öllu í GitHub skýið.

Uppsetning á GitHub

Ég bjó til nýtt repository á GitHubinu mínu sem heitir Tolvustudd-Framleidsla. Síðan opnaði ég git bash í local möppunni sem innihélt skjöl síðunnar. Svo keyri ég þessa skipun til að geta fikrað mig áfram github megin.

Félagi minn og samnemandi hann Sindri Dagur er flinkur í GitHub svo hann ráðlagði mér að nota eftirfarandi skipanir til að koma local foldernum á GitHub: 

    
    git clone https://github.com/SiggiPatrik/Tolvustudd-Framleidsla
    git add .
    git commit -m "Setja template files inn"
    git push

Þetta bara svínvirkaði í fyrstu tilraun svo ekki þurfti að hafa meiri áhyggjur af því. Síðustu þrjár línurnar þarf svo að framkvæma í hvert sinn sem það á að vista breytingar úr local folder yfir í GitHub repositoryið. Að sjálfsögðu er commentið ekki það sama að hverju sinni samt, heldur það sem er viðeigandi í hvert skipti. Ef ég ætla mér svo að vinna á nýrri tölvu eða vil vera viss um að ég sé að vinna með sama file og er á GitHub þá finn ég mér folder sem ég vil vinna í og geri 'git clone' þar, annars duga síðustu þrjú skrefin til að uploada aftur á github.

Breytingar á síðunni

Ég eyddi miklum tíma í að dunda mér við að snyrta sniðmátið og setja inn suma hluti og fjarlægja aðra. Ég nýtti mér fyrri þekkingu á HTML til að fikta mig áfram, frekar en að googla eða afrita frá öðrum. Ég mun líklegast aldrei klára það alveg en vonandi get ég talið útlit vefsíðunnar ásættanlegt að lokum.
Á þessari síðu fyrir verkefni 1 var raunar ekkert til að fjarlægja nema texti. Því eyddi ég fullt af lorem ipsum út og setti minn eiginn texta í staðinn, ásamt myndum.

Myndir og skjáskot

Að hlaða myndum á síðuna er frekar einfalt. Það þarf bara að hafa myndina vistaða inni í möppunni sem GitHub skoðar og vísa til þeirra með viðeigandi hlekk. Hins vegar er sniðugt að hafa góðan skjalastrúktúr á myndunum sínum, t.d. hafa 'images' möppu og svo sér möppu fyrir hverja síða þar undir. Fyrir fyrsta verkefnið (þessa síðu) gleymdist það en fljótt sá ég hversu mikla ringulreið það skapaði. Því skipulagði ég 'images' möppuna mína aðeins betur en það þýðir að þá þarf ég handvirkt að uppfæra sérhvern hlekk á myndirnar sem voru ekki vistaðar á réttum stað.
Á flestum stöðum kaus ég að hafa myndirnar 'centered' en til þess nýtti ég mér klasann sem fylgdi með CSS skránni. Með hjálp hans bý ég til div sem inniheldur mynd svona: 

  
    <div class="row aln-center">
        <img src="images/..." width="..." alt="...">
    </div>

Væntingar mínar til áfangans

Ég vil að þessi áfangi hjálpi mér að skipuleggja og framkvæma nett og krefjandi verkefni sem ég get svo sett fram á fagmannlegan hátt á þessari síðu og vonandi notað hana sem part af starfs- og skólaumsóknum. Ég vil leggja mig fram í að gera þessi verkefni vel og sérstaklega að greina vel frá þeim svo þessi síða nýtist sem fjöður í hattinum mínum í framtíðinni.
